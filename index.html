Technologies

Server Side

Babel

Transpiled latest ECMAScript to browser supported version

TLS/SSL

Online security protocol to ensure no one like ISPs are listening to, or meddling with your data transmission.

Using Lets Encrypt, the new, free, automated way to do ssl certificates for security

NodeJS

Webserver of choice, designed for web development

Client Side

WebRTC API

This is a very experimental technology currently only available on the latest Firefox and Chrome browsers, with spotty support on others

MediaRecorder API

This allows recording a media stream and creating a webm video file

CCV

This is a facial recognition library with a cascade to detect faces in images

ES2015

This is the latest javascript built in to the latest browser. This can be transpiled back to ES5 by a transpiler like babel for backward compatibility
